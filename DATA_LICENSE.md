# 数据许可策略

本文件定义 `candidate-0.2.0-vertical` 的公开数据许可。它仅许可项目贡献者独立整理的数据库与文字，不转授课程标准、教材或教辅的任何权利。

| 层 | 计划许可/规则 |
| --- | --- |
| 图谱数据库 | [ODbL 1.0](https://opendatacommons.org/licenses/odbl/1-0/)。涵盖节点集合、稳定 ID、关系与标准映射的数据库结构。 |
| 项目自有文字 | [CC BY-SA 4.0](https://creativecommons.org/licenses/by-sa/4.0/legalcode)。涵盖项目编写的名称、描述、掌握证据、检测提示、依赖理由及领域摘要。 |
| 课程标准与教材 | 保留上游权利与条款，不以本项目名义重新许可。默认只发布可定位的标识、页码、书目信息和自有的简短索引标题。 |
| 代码 | MIT License，见 `LICENSE`。 |

本项目不在未确认上游授权前复制课程标准原文、教材扫描页、教材 OCR 全文、习题全文或答案。详见 `PROVENANCE.md` 和 `docs/SOURCE_POLICY.md`。

使用或再发布数据时，必须保留本文件、`PROVENANCE.md`、`CITATION.cff` 和适用的署名/相同方式共享义务。第三方来源的上游条款优先；来源无法定位或版权存在争议的数据不得进入候选发布。
