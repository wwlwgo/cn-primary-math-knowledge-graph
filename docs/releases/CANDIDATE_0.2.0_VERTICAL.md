# candidate-0.2.0-vertical 发布说明（准备稿）

## 覆盖范围

本候选版本覆盖北师大版一年级上、一下、二年级上、二年级下和三年级上教材映射：91 个 `reviewed` 学习目标，19 条 `soft` 依赖，56 条课程标准登记。

节点包含课程标准 ID、教材书目信息与印刷页码、两条可观察证据和诊断提示。所有数据来自维护者的私有教材视觉核验与独立整理，不包含教材扫描件、原文、OCR、题目或答案。

## 证据层级

这是 `candidate` 数据，不是专家确认版：

- 所有边均为 `soft`，表示有帮助但存在替代学习路径；
- 未发布 `hard` 边，未存在 `approved` 节点或边；
- 教材版次未在部分封面中显示，来源记录未臆造年份或版次；
- 不应将教材先后顺序解释为认知必要关系。

## 已知限制

- 当前只使用北师大版，未完成跨版本稳定性验证。
- 三上使用的是承接扩展，不宣称覆盖完整第二学段。
- 标准第二、三学段若为内容要求集合，映射仅表明课程相关性，不等同于一对一官方微知识点。
- 诊断提示是检查入口，不是正式试题或标准化测验。

## 后续人教版交叉检验队列

优先接收和映射：人教版一年级上下册、二年级上下册、三年级上册。对每个候选节点记录 `supports`、`different-wording`、`different-granularity`、`not-located` 或 `conflicts`，并将冲突用于节点粒度和表述审查，不以教材编排顺序证明依赖。

正式发布前仍须完成公开许可证文本、GitHub Release 工件和社区复核入口配置。
