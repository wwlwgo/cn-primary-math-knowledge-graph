# Marble Skill Taxonomy 审读记录

审读对象：[withmarbleapp/os-taxonomy](https://github.com/withmarbleapp/os-taxonomy)，审读日期：2026-07-19，提交：`96a7933`。

## 已核验的做法

该项目发布了 1,590 个节点、3,221 条依赖、3,261 条课程标准和 183 个领域摘要。仓库同时提供 JSON Schema、无第三方依赖的完整性校验器、文件 SHA-256、来源说明、版本日志和 `CITATION.cff`。其节点带学习证据和标准映射；边区分 `hard`/`soft` 并解释原因；图谱定位为纯数据基础设施。

## 本项目采纳

- 保持纯 JSON 数据、独立 Schema 和无运行时依赖的校验器。
- 对节点、依赖和标准映射做引用完整性检查，并在清单中保存文件校验和。
- 分离数据库、项目自有文字与第三方标准的许可和来源说明。
- 维护版本日志、引用元数据和可复现的发布清单。

## 本项目不直接沿用的部分

- 不复制 Marble 的数据、文本、关系、媒体或课程标准映射；它是方法参考，不是数据源。
- 中国课程标准和教材的再发布条件尚未确认，因此默认仅公开标识、定位和自有短标题，不公开原文。
- Marble 的 Schema 将边的 `reason` 设为可空，且未要求边来源或审核状态；本项目要求每条边均有理由、来源和审核状态，并规定 `hard` 边双人教育审核。
- Marble 的节点年级采用近似年龄范围；本项目以中国学段/年级为主，并保留年龄作为可推导的展示层，避免把地区性入学年龄硬编码为事实。
