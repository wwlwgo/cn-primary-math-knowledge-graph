## 复核规则

1. 先确认目标是否是稳定可诊断能力，而非教材特有活动。
2. 仅登记版本代号、册次、页码/章节定位和独立结论。
3. 结果只能是：直接支持、表述/粒度差异、当前范围未定位、冲突待裁决。
4. 未定位不是反证；冲突不会自动改变节点或依赖强度。
5. 拟议变更须使用社区复核模板，并由维护者记录采纳、拒绝或待定理由。
