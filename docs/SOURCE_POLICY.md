# 来源与版权政策

本项目公开的是独立整理的学习目标、关系、短描述和书目信息。它不公开教材扫描件、教材原文、逐页 OCR、习题全文、答案或其他受限材料。

每条来源记录必须包含来源类别、可读引文和定位信息。教材通常至少记录出版社、书名、版次、册次、年份和页码；课程标准至少记录文件名、版本、章节或条目。无法公开访问的来源可以被引用，但其内容不得复制到仓库。

《义务教育数学课程标准（2022年版）》的规范性入口固定为教育部 `教材〔2022〕2号` 的[中国政府网发布通知](https://www.gov.cn/zhengce/zhengceku/2022-04/21/content_5686535.htm)。正式节点不得仅标注“课标”或只引用文档级 ID，必须引用 `data/standards.json` 中已登记、可定位的具体条目。

模型输出是候选信息，不是来源。任何由模型提出的节点或边都必须补充独立来源并完成审核。来源存在争议、版权不明或无法定位时，相关数据不得进入 `approved` 状态。

不同出版社教材可作为交叉检验来源。教师用书、教辅材料、教学设计和研究材料可使用 `teacher_guide` 或 `research` 类别记录，但只能记录独立撰写的短引文、书目信息和页码/章节定位；不得录入其受版权保护的原文、题目、答案、扫描件或 OCR 全文。教辅材料属于辅助证据，不能替代课程标准的精确映射，也不能单独把候选依赖升级为 `hard`。

对于公开的 `candidate` 数据，来源完整和机器校验通过是最低要求；只有在满足相应人工审核门槛后，数据才可标为 `approved` 并纳入 `expert-verified` 发布范围。
