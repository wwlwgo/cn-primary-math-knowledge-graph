# Changelog

本项目遵循语义化版本。候选版本不是专家确认版本。

## candidate-0.3.0-cross-edition - 2026-07-23

- 扩展至一上至三下六册候选范围：95 个 `reviewed` 节点、18 条 `soft` 依赖、19 个课程标准能力锚点和 56 条课标登记。
- 完成双版本私有核验范围内的跨版本重算：能力锚点重合率为 18/19，严格可诊断节点直接确认数为 79/87。
- 新增三年级下册跨版本对照报告，并将复核队列更新为 27 项全部闭环、0 个开放项。
- 保持候选层级：未新增 `hard` 边、`approved` 数据或专家确认结论；公开树仍不包含教材原文、扫描页、OCR、题目、答案或私有输入。

- 新增候选发布复现、许可证、审计和 GitHub Release 工件。
